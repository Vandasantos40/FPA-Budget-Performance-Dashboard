# 📊 FP&A Budget Performance Dashboard

## 📌 Project Overview

This Power BI dashboard provides an executive view of budget performance by comparing planned budgets against actual results across multiple departments. The dashboard highlights financial variances, budget achievement, and monthly performance trends, supporting management with actionable insights for decision-making.

---

## 🎯 Business Objective

The objective of this dashboard is to support FP&A (Financial Planning & Analysis) activities by:

- Monitoring Budget vs Actual performance
- Identifying favorable and unfavorable variances
- Tracking budget achievement across departments
- Supporting executive reporting with financial KPIs
- Enabling interactive analysis by month and department

---

## 📈 Dashboard Features

- Executive KPI Cards
- Budget vs Actual Analysis
- Budget Variance by Department
- Monthly Budget Performance Trend
- Budget Achievement %
- Executive Financial Insights
- Interactive Slicers (Department & Month)

---

## 📊 Key Performance Indicators (KPIs)

- Total Budget
- Total Actual
- Total Variance
- Budget Achievement %
- Variance %

---

## 🛠 Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## 💼 Business Value

This dashboard enables finance teams to monitor financial performance, compare actual results against planned budgets, identify departments requiring attention, and support executive decision-making through clear and interactive reporting.
